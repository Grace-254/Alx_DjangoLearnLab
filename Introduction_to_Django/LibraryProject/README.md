# 0. Introduction to Django Development Environment Setup

This directory contains the `LibraryProject` Django project for the **Introduction_to_Django** task.

## Project Setup

- Python virtual environment created locally (not required in this repository).
- Django installed using `pip install django`.
- Project created with `django-admin startproject LibraryProject` inside the `Introduction_to_Django` directory.
